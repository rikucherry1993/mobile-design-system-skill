Icon set

Inline-SVG sprite, themed by tokens — fill:none; stroke:currentColor; stroke-width:var(--icon-stroke). Adjust --icon-stroke / --icon-size in styles.css to re-theme every icon at once (thin SF-like vs heavier Material). Original geometry drawn in the style of classic open sets (Lucide/Feather, Material Symbols). Use: <svg class="ico"><use href="#i-home"/></svg>. Never use emoji for UI icons.
