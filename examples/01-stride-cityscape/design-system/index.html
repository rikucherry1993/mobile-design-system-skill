Stride Design System

A warm, friendly city-fitness system. Muted teal brand + warm city-dusk illustration palette, soft surfaces, large radii, rounded display type. The signature is a wide city-skyline blob illustration (token-driven SVG). Every fill ships a contrast-checked on-* partner; source of truth is styles.css. Dev Mode inspects any component; ⤓ Install design system scaffolds/updates it all.

Color

Material 3 roles + an expression palette for the illustrations. Each fill swatch shows its paired on-* (top-right) — WCAG-verified. Neutrals tinted warm; teal↔amber are a complementary pair.

Typography

M3 type scale — rounded Nunito for display/headline/title, Inter for body/label.

Spacing

4dp grid + fine --opt-* optical steps.

Shape

M3 shape scale — soft, with large --r-xl (28) for cards & illustration heroes.

Elevation

M3 levels 0–5 as soft, warm-tinted shadows.
