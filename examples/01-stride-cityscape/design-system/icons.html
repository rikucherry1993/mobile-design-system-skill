Icons

Token-driven inline SVG sprite: stroke:currentColor; stroke-width:var(--icon-stroke), so the set reskins with the theme (Material/outlined, 1.9 stroke). Use with <svg class="ico"><use href="#i-run"/></svg>. Never emoji.

Illustrations

The signature: wide flat city-skyline blob illustrations as inline SVG. The blob gradient and skyline silhouette are token-driven — set --illus-1 / --illus-2 / --illus-sil via .illus--day / .illus--dusk / .illus--teal, so they re-theme with everything else. Each instance gets unique gradient ids (helper below). Use them wide, edge-to-edge, as screen heroes.
