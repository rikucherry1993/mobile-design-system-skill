Margin

A monochrome, hand-drawn system for a reading & notes app — read, listen, highlight, and keep marginalia. Thick ink outlines, hard offset shadows, heavy display type, hand-drawn line illustrations, a handwriting face for notes. Source of truth is styles.css. Dev Mode inspects; ⤓ Install design system scaffolds it all.

Color

Monochrome — ink on warm paper, plus an inverse ink card. Each fill shows its paired on-* (top-right), WCAG-verified. "Status" is shown by treatment (raised / dashed / inverted), not hue.

Typography

Heavy Space Grotesk display, Inter body, Lora serif for reading, Caveat handwriting for notes.

Spacing

4dp grid + fine --opt-* steps.

Shape

Large radii + pills, all drawn with a 2px --stroke ink outline.

Elevation

Mostly flat; the signature is a hard offset shadow (--shadow-hard) for a sticker feel.
