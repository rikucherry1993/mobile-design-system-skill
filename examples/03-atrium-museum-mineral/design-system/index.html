Atrium Design System

A calm, mineral system for a museum & gallery guide — a palette in the spirit of Le Corbusier's architectural keyboard (slate, ochre, plaster, umber). Every fill ships a contrast-checked --on-* partner; the source of truth is styles.css. Toggle Dev Mode to inspect, or ⤓ Install design system to scaffold/update it all.

Colors

Semantic roles on a light plaster base — each fill shows its paired --on-* (top-right), WCAG-verified. The last three are exhibition tones (.tone-navy / .tone-clay / .tone-sage): per-page background colours that let each exhibition harmonise with its artwork (deep navy for dark works, terracotta for warm, sage for green). See the concept.

Typography

Editorial Spectral serif for display, clean Inter for text, IBM Plex Mono for figures — a catalogue-like hierarchy.

Spacing

One 4dp-grid scale, plus fine --opt-* optical nudges.

Elevation

Soft, ink-tinted, layered shadows — restrained for a calm, architectural surface.

Use them as full-bleed heroes, exhibition cards, and ticket thumbnails. Swap in real public-domain images by replacing the SVG with an <img> in the same .artwin.
